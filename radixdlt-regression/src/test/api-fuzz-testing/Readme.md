### Setting up python environment

```bash
#Install pyenv using brew
brew install pyenv-virtualenv

#Install pyenv virtual environment using brew
brew install pyenv-virtualenv

#Check pyenv versions using below command
pyenv versions

#Install python version 
pyenv install -v 3.7.2

#Create pyenv environment using below command
pyenv virtualenv 3.7.6 radixdlt-fuzzer

#Setup the local environment of the project to be use as below
pyenv local radixdlt-fuzzer

# Check the version of python on the terminal 
python -V
# .....Python 3.7.6
````
